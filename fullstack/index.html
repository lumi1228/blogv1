GitHub

对标阿里 P6，从前端工程师直达大前端；一个业务场景深度打通全端 + 全栈 + 全流程，覆盖 PC、管理后台、WebApp、uniapp 小程序、移动 App、跨端桌面应用与服务端接口。

Fullstack Atlas · 战略视图

全端 · 全栈 · 全流程

通过统一业务场景串联多端形态与服务端能力，聚焦工程化、流程化与架构思维，打造可复用的企业级解决方案。

以下目录按阶段推进，保留原有学习路径，并配合视觉增强，便于快速定位。

目录

├── 阶段一：解锁“大前端思维”                     # 入门与通识
│   ├── 认识大前端，全栈开发初体验                # 建立全局视角
│   ├── 企业标准的开发环境搭建                    # 工程化与规范化
│   ├── NoSQL 数据库的设计与集成                  # 非关系型数据存储
│   └── JWT 登录鉴权（通用登录模块后端）          # 认证与会话
│
├── 阶段二：项目全流程 DevOps                     # 团队与流程
│   ├── 文档管理与缺陷控制                        # 协作与质量
│   ├── 版本管理                                  # Git 流程
│   └── 自动化流程                                # CI / CD
│
├── 阶段三：PC 端进阶前后端开发                   # 业务模块实战
│   ├── 首页模块                                  # SSR / SEO 可选
│   ├── 用户中心                                  # 账户与资料
│   ├── 发帖 / 回帖模块                            # 帖子流与交互
│   └── 消息中间件（WebSocket 通信）              # 实时通信
│
├── 阶段四：CMS 后台管理系统开发                  # 中后台体系
│   ├── 组件化思想进阶：iView 上手文章管理        # 组件库实践
│   ├── 前端权限设计方案                          # 鉴权与路由守卫
│   ├── 基于角色与菜单的权限设计                  # RBAC / 菜单
│   └── 首页可视化图表与日志服务                  # 图表与审计
│
├── 阶段五：WebApp 开发与 Vue3 项目重构          # 移动 Web 与重构
│   ├── WebApp 页面设计与开发                     # 适配与交互
│   ├── Vue3 全家桶 + Vite 开发工具              # 生态与构建
│   ├── 前后端项目 Vue3 重构                      # 迁移策略
│   └── 性能优化（服务端渲染）                    # SSR 性能
│
├── 阶段六：小程序开发与支付专题                  # 小程序 / 支付
├── 阶段七：跨端开发（移动端跨平台与桌面端）      # 跨端与原生

项目架构
